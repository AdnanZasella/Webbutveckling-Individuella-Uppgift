# MyShop Accessibility Project

## Beskrivning
Detta är en produktlista-sida där tillgänglighet och prestanda har förbättrats.

## Förbättringar
- Förbättrad semantisk HTML-struktur
- Förbättrade formulär med labels
- Alt-texter på alla bilder
- WCAG färgkontrastfixar
- LCP prestandaoptimeringar
- Axe accessibility testing

## Hur man kör projektet
1. Klona repot
2. Öppna index.html i Live Server

## Tekniker
- HTML
- CSS
- JavaScript
- Axe accessibility testing
